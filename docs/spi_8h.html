Detailed Description

Header file for SPI (Serial Peripheral Interface) communication in Rishka applications.

Author
Nathanne Isip

This header file defines the SPI class, which provides functionalities for SPI communication on ESP32-WROVER microcontrollers.

enum spi_mode_t

Enumeration for SPI data modes.

The spi_mode_t enumeration defines different SPI data modes that control the clock polarity and phase during SPI communication.

Enumerator
SPI_MODE0 

Clock polarity (CPOL) = 0, Clock phase (CPHA) = 0

SPI_MODE1 

Clock polarity (CPOL) = 0, Clock phase (CPHA) = 1

SPI_MODE2 

Clock polarity (CPOL) = 1, Clock phase (CPHA) = 0

SPI_MODE3 

Clock polarity (CPOL) = 1, Clock phase (CPHA) = 1
